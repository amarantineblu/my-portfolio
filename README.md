# My Portfolio 🚀

A modern, responsive portfolio website built with **React + TypeScript + Vite**.  
It showcases my projects, experiences, and provides an admin dashboard protected by Firebase Authentication.

---

## ✨ Features
- 🏠 **Guest Layout** with pages:
  - Home
  - About
  - Experiences
  - Projects
  - Contact
- 🔐 **Authentication**
  - Register and Login with Firebase Auth
  - Protected Admin Dashboard
- 📂 **Admin Layout**
  - Dashboard page (accessible only to authenticated users)
- ⚡ **Fast Development** with Vite
- 🎨 Custom styling with CSS

---

## 🛠️ Tech Stack
- **Frontend:** React, TypeScript, Vite
- **Routing:** React Router v6
- **Auth:** Firebase Authentication
- **Database (optional):** Firebase Firestore (future expansion)
- **Deployment:** GitHub Pages / Vercel / Netlify
